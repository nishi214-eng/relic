# オシキー

<p align="center"><img src="./static/logo/logo.PNG" alt="ロゴ" style="width:80%"/></p>  
オシキーは、ユーザーがアニメや漫画などの「推し」に関わる場所を投稿することで、「推し活による経済の活発化・地方創生」や「地域で見つけた隠れた名作コンテンツゆかりの地の共有」、「推し活コミュニティの形成」を目的としたユーザー投稿型のWebマッピングサービスです。  

(第2回Relicプログラミングコンテスト応募作品)

## 特徴・工夫点
1. 画像によるピンの投稿  
   その場で写真を撮り、その画像をアップロードするだけで簡単にマップ上にピンを立てることが出来ます。  
   ※画像にExif情報(位置情報)が含まれている必要があります
2. ピンの情報確認  
   説明を見たいピンをクリックすることで住所や説明を見ることが出来ます。
3. ピンの絞り込み機能  
   登録したタグやピンにより、表示するピンを絞り込むことが出来ます。
4. 独自のキャラクターUI  
   このサイト上でも「推し」が作れるようなデザイン。

## 開発背景
和歌山は「サマータイムレンダ」や「AIR」など多くの名作コンテンツの舞台となっています。  
本アプリの企画者は、和歌山を始めて訪れた際に、町の端々にコンテンツに関連したアイテムを見つけたとき感動を覚えました。ですが、そんな日常の中にあるコンテンツを共有することができない、自分の他に共有している人を見つけられないといった課題を日々感じています。この課題のために、コンテンツへの欲求、所謂「パブリックサーチ欲」を満たせない人は多くいる、と私達は考えました。よってグッズ販売店や聖地の情報を視覚的にわかりやすくまとめると共に、日常の些細なコンテンツとの出会いを「オシキー（推し紀伊）」として共有するプラットフォームを制作したのです。
